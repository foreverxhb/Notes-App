# Notes-App
A text-based notes manager with file read/write.  Java File I/O

---

# Features

-  Add new notes
-  View all saved notes
-  Stores notes in `notes.txt`
-  Uses standard Java I/O libraries
-  Beginner-friendly code structure

---

# How to run

- javac Notes.java
- java Notes
